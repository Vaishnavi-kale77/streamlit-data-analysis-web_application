# Streamlit Data Analysis Web App

This project is a Streamlit-based web application for automated data analysis, preprocessing, and machine learning.

## Features:
- Upload dataset (CSV)
- Data cleaning (missing values, duplicates)
- Outlier detection
- Data visualization (Plotly charts)
- Machine learning model training & evaluation

## Technologies Used:
- Python
- Streamlit
- Pandas, NumPy
- Scikit-learn
- Plotly

## How to Run:
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Purpose:
To simplify data analysis and provide an interactive ML workflow for users.
